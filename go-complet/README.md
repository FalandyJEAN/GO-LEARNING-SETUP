# Cours Complet de Go

Bienvenue dans ce cours complet pour apprendre le langage Go de A à Z !

## Structure du cours

- `01-introduction/` : Présentation, installation, premiers pas
- `02-bases/` : Variables, types, opérateurs, conditions, boucles
- `03-fonctions/` : Fonctions, erreurs, portée
- `04-structures/` : Structs, méthodes, interfaces
- `05-collections/` : Tableaux, slices, maps
- `06-pointeurs/` : Pointeurs, références
- `07-concurrence/` : Goroutines, channels, mutex
- `08-fichiers-et-io/` : Fichiers, entrées/sorties
- `09-tests/` : Tests unitaires, benchmarks
- `10-projets/` : Exercices pratiques et mini-projets

Chaque dossier contiendra des leçons, des exemples de code et des exercices.

---

Commencez par le dossier `01-introduction` !
