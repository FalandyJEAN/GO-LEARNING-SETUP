# 10. Projets Pratiques Go

## Objectifs pédagogiques
- Mettre en œuvre toutes les compétences acquises
- Construire des applications réelles, du CLI au backend

---

## 1. Idées de projets
- Calculatrice CLI avancée (gestion des erreurs, tests)
- Gestionnaire de contacts avec sauvegarde JSON
- Serveur HTTP de gestion de tâches (CRUD)
- API RESTful pour gestion d’utilisateurs
- Crawler web concurrent
- Analyseur de logs système
- Mini-backend complet (authentification, base de données, API)

## 2. Conseils méthodologiques
- Découpez le projet en modules/fonctions
- Testez chaque partie séparément
- Documentez votre code
- Utilisez git pour le suivi

## 3. Ressources complémentaires
- [Awesome Go](https://awesome-go.com/)
- [Go Patterns](https://github.com/tmrts/go-patterns)
- [Go Web Examples](https://gowebexamples.com/)
